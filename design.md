## 🍽️ MealSignup - 家族向け食事申し込みアプリ

### **プロジェクト概要**
- 家族向け食事申し込みアプリ 
- React + TypeScript (フロントエンド)
- Node.js + TypeScript (バックエンド)
- Clerk認証 (Google/Apple/メール認証)
- Fly.ioでデプロイ予定

### **コア機能**

#### 🏠 **家族管理機能**
- **家族作成**: 新しい家族グループを作成
- **招待システム**: 招待コードで家族メンバーを追加
- **メンバー管理**: 家族構成員の一覧表示・編集
- **権限管理**: 管理者（親）と一般ユーザー（子供）

#### 📅 **食事申し込み機能**
```
今日の食事予定
┌─────────────────┐
│ 田中太郎さん     │
│ □ 朝食が必要    │
│ □ 昼食が必要    │  
│ □ 夕食が必要    │
│ [記録する]      │
└─────────────────┘
```

#### 📊 **食事予定表示**
- **今日の予定**: 家族全員の本日の食事予定
- **週間ビュー**: 1週間分の食事申し込み状況
- **カレンダー**: 月間での食事予定確認
- **統計**: 食事回数の集計

### **ユーザーフロー**

#### 🔐 **認証フロー**
```
1. ログイン画面
   [Googleでログイン] [Appleでログイン]
   ──────────────────
   メール: [________]
   パスワード: [____]
   [ログイン] [新規登録]

2. 初回ユーザー
   → 家族作成 or 家族参加選択

3. 既存ユーザー  
   → 食事申し込み画面へ
```

#### 🏡 **家族設定フロー**
```
新規家族作成:
家族名入力 → 招待コード生成 → メンバー招待

既存家族参加:
招待コード入力 → 承認待ち → 参加完了
```

### **画面構成**

#### 📱 **メイン画面**
1. **ホーム** - 今日の食事状況サマリー
2. **申し込み** - 個人の食事申し込み
3. **家族予定** - 全メンバーの食事予定
4. **設定** - プロフィール・家族管理

#### 👨‍👩‍👧‍👦 **家族向け特化機能**
- **シンプルUI**: 子供でも使いやすい大きなボタン
- **アイコン設定**: 家族メンバーのアバター
- **通知機能**: 申し込み忘れのリマインド（将来）
- **履歴機能**: 過去の食事記録確認

### **将来の拡張機能候補**
- 🔔 **通知システム**: 申し込み締切のリマインド
- 🛒 **食材管理**: 必要な食材の自動計算
- 📈 **分析機能**: 家族の食事パターン分析
- 🏢 **企業版**: 社食・仕出し弁当の申し込み
- 🍱 **メニュー管理**: 日替わりメニューの表示
